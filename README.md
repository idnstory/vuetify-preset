# `@haezoom/vuetify-preset`

The Haezoom preset for Vuetify.

## Install

```
$ npm install @haezoom/vuetify-preset
```

## Usage

```js
import Vue from "vue";
import Vuetify from "vuetify/lib";
import { preset } from "@haezoom/vuetify-preset/preset";

Vue.use(Vuetify);

export default new Vuetify({
  preset,
  ...
});
```
